I had been postponing my goal to learn my target language (russian) because of school work. So, as a solution, I'm developing 
an MCP server that will connect to Folk, Claude, GPT, or any LLM that has MCP config to create a daily digest for me which will include things like:


### Word of the Day
Folk will give me a word every day, including its gender and use cases, from a video it pulls up daily.
It pulls up a different video every day from a curated list of channels I've made. 


### Video to Watch
Folk will give me a video to watch, whether it's for grammar or street convo.

### Video Transcript 
Folk will pull up the transcript of the video which is useful for reading daily.


### Translate a word
Translate a word if I need to


### Conversation Drill
After the daily digest, Folk will start a conversation with which I'll engage.
